Recent Projects

Kitchen Counter

https://profound-moonbeam-1e2506.netlify.app

Kitchen Counter is a food inventory application. At a minimum, it tracks what foods a user currently has in their cupboard or refrigerator. It also permits a user to record what they eat. With these two pieces of information the user will know when they are running low on a food item. Their food shopping will thus be more efficient and will eliminate those trips to the supermarket because one has forgotten that one product they meant to buy.

Technologies employed: Javascript, React.js, Mongoose, Node.js, Express, MongoDB, Heroku, Netlify, HTML5, CSS and Animate.css
Github repositories: https://github.com/Gingaling/eat-n-shop-frontend (frontend); https://github.com/Gingaling/eat-n-shop-api (backend)

Book Roulette

https://book-roulette.netlify.app/

Book Roulette is a book recommendation app that generates a random book to users based on genre preferences. Each user has the capability to add books that they would recommend to fellow users as well.

Technologies employed: Javascript, React.js, Mongoose, Node.js, Express, MongoDB, Heroku, Netlify, Axios, HTML5 and CSS
Github repositories: https://github.com/GeMMS-Project-2/book-roulette-frontend (frontend); https://github.com/GeMMS-Project-2/book-roulette-backend (backend)

A Trio of Cats, Less Two

https://xenodochial-heisenberg-1997c1.netlify.app/

A Trio of Cats, Less Two is a single-page application created with ReactJS and styled utilizing Flexbox. The app appropriately uses the useState hook, a new feature in the most recent React upgrade. The app incorporates the use of a fetch operation to a third party API and uses retrieved data. In this case, it was random cat images. Further the app demonstrates the use of calling a component with state via an onClick event.

Technologies employed: Javascript, React.js (including React Router and ReactDOM), JSX, vanilla Javascript, Flexbox and Netlify
Github repository: https://github.com/Gingaling/a-trio-of-cats-3
